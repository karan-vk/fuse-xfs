# Contributing to fuse-xfs

Thank you for your interest in contributing to fuse-xfs!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/fuse-xfs.git`
3. Install macFUSE: `brew install --cask macfuse`
4. Build the project: `cd src && make`

## Making Changes

1. Create a new branch for your feature or fix
2. Make your changes
3. Test your changes with real XFS filesystems
4. Submit a pull request

## Reporting Issues

Please include:
- macOS version
- Apple Silicon or Intel
- XFS filesystem version (V4 or V5)
- Error messages or logs

## Code Style

- Follow existing code style
- Comment complex sections
- Update documentation for new features
